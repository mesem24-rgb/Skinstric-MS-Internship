# Skinstric AI Demographics Experience

A high-fidelity frontend application inspired by the Skinstric design experience, focused on responsive UI engineering, polished interactions, animated transitions, and production-style user flows using Next.js and React.

# Overview

This project recreates a modern AI-powered demographic analysis experience with a strong emphasis on frontend architecture, responsive layouts, UI polish, and interactive user experience design.

The application includes animated onboarding flows, camera permission handling, dynamic loading states, responsive transitions, and component-driven UI patterns designed to closely match a production-level design reference.

The project was built in multiple development phases focused on:
- UI foundation
- Responsive implementation
- Camera workflow integration
- Loading state improvements
- UX polish and interaction refinement
- Production-style frontend architecture

# Challenges & Solutions

## Challenge:
Maintaining responsive layout consistency while recreating a highly polished design reference across multiple screen sizes and interaction states.

## Solution:
Implemented reusable layout structures, responsive breakpoint adjustments, animated UI transitions, and conditional rendering patterns to ensure a smooth and consistent user experience across desktop and mobile devices.

Additional focus was placed on:
- Camera permission handling
- Loading state feedback
- Modal interactions
- Dynamic responsive positioning
- UI spacing and typography parity

# Development Process

The project was developed iteratively in phases to better simulate a real-world frontend workflow.

## Phase 1 — Foundation
- Built initial layout structure
- Implemented responsive sections
- Created reusable components
- Established routing and styling architecture

## Phase 2 — Camera & Flow Integration
- Added camera/selfie workflow
- Built demographic summary flow
- Implemented loading states
- Improved route transitions and state handling

## Phase 3 — UI Polish & Refinement
- Refined hover interactions
- Improved responsive breakpoints
- Enhanced animation timing
- Added skeleton loading improvements
- Matched production-style UI alignment and spacing
